MCP Server 工具说明

MCP Server 提供以下工具，AI 可自动调用这些工具完成任务：

🌤️ weather - 天气查询

查询指定城市的天气信息，包括温度、湿度、天气状况等

参数：city 城市名称（北京、上海、广州、深圳、杭州）

🧮 calculator - 计算器

执行数学表达式计算，支持加减乘除、括号、幂运算

参数：expression 数学表达式

🗄️ database - 数据库查询

查询用户信息，包括姓名、邮箱、会员等级、账户余额

参数：userId 用户ID（1001-1004）

API 接口说明

GET/api/chat?message=xxx

GET 方式调用聊天接口，参数通过 URL 传递

POST/api/chat

POST 方式调用聊天接口，Body: {"message": "xxx"}
